# WIP
This is a work in progress game engine using OpenGL.
## Dependencies
### Build tools
- GCC or Clang
- GNU Make
- CMake or pkg-config
### Libraries
- GLFW or SDL2
- libconfig
## Building
### Using make
`$ ./configure.sh` and `$ make`
### Using CMake
```
mkdir build
cd build
cmake ..
make
```

SDL2 backend may not work correctly.
